# OMV-NAS-Rpi4b Homelab Documentation

## Overview

This repository documents the setup and operation of a Raspberry Pi based NAS system using OpenMediaVault.

The goal of this documentation is to provide a **living technical documentation** including:

- System architecture
- Hardware configuration
- Storage layout
- Docker services
- Network configuration
- Troubleshooting knowledge base
- Best practices

---

## System

Hardware:

- Raspberry Pi 4B
- FireCuda 5TB HDD
- Samsung SSD
- SanDisk NVMe

Software:

- OpenMediaVault 8
- Docker
- JDownloader
- Pi-hole (planned)
- Unbound (planned)
